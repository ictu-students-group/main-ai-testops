# main-ai-testops
Mục đích của kho: 
- Thảo luận để thu thập các ý tưởng, các yêu cầu chung cho dự án xây dựng mô đun sinh test cases tự động sử dụng LLM cho các bài kiểm tra thực hành Java.
- Chỉ chứa requirements & analysis, không chứa code
  Kho này sẽ quản lý:
  - Stakeholder's requests (NEED)
  - Product features (FEAT)
  - Use cases (UC)
  - Supplementary Requirements (SR) 
Tất cả các yêu cầu được quản lý thông qua GitHub issues. Traceability được duy trì thông qua issue linking.
